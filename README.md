# Python Calculator

A simple command-line calculator written in Python.  
This project demonstrates basic Python programming concepts such as functions, user input handling, error handling, and arithmetic operations.  
It was created as part of my career comeback to showcase coding skills.

---

## Features

- Addition (`+`), Subtraction (`-`), Multiplication (`*`), Division (`/`)  
- Supports floating-point numbers for precise calculations  
- Handles division by zero gracefully  
- Checks for invalid numeric input and prevents crashes  
- Clean, professional command-line interface

---

## Requirements

- Python 3.x
